# 🌸 Bliss & Petals

**Bliss & Petals** is a modern, elegant flower boutique website designed to provide a seamless online shopping experience for floral lovers. The project showcases a premium collection of handcrafted bouquets with a clean interface, responsive layout, and aesthetically pleasing design.

The website focuses on simplicity, elegance, and user-friendly navigation while highlighting beautiful floral arrangements for different occasions.

---

## ✨ Features

* 🌹 Elegant homepage with a premium floral theme
* 🛍️ Product catalog showcasing handcrafted bouquets
* 📄 Individual product pages with detailed descriptions
* ⭐ Product ratings and pricing
* 🧭 Responsive navigation bar with dropdown menu
* 🎨 Soft pastel color palette and modern typography
* 📱 Responsive design for different screen sizes
* 💐 Beautiful product cards with hover effects

---

## 🛠️ Built With

* HTML5
* CSS3
* Google Fonts (Poppins, Inter, Imperial Script)
